  - ### <span style="color:yellow">机制2：没有真正意义上的家人 </span> 开始机制后，角色脚下会有金色探测仪；跟着指向的方向，引导BOSS攻击对应的地板；<img src="./img/娅内默2阶段/没有真正意义上的家人1.png" alt="没有真正意义上的家人1">完成机制，地板会变成紫色；带黑色的污染地板不要碰，会掉血；<img src="./img/娅内默2阶段/没有真正意义上的家人2.png" alt="没有真正意义上的家人2">出现台词：【你怎么还是如此执迷不悟】时，找到金色地板位置，站里面躲避后续爆炸；<img src="./img/娅内默2阶段/没有真正意义上的家人3.png" alt="没有真正意义上的家人3">完成三次探测后，机制成功

  - ### <span style="color:yellow">机制3：不要碰我的目标 </span> 地图中会出现带绿圈的金色小正方体和大型正方体；拾取小正方体或者进入大正方体都可以躲避全屏爆炸；<img src="./img/娅内默2阶段/不要碰我的目标1.png" alt="不要碰我的目标1">但是不能两个都弄，不然队友就会遭殃。

  - ### <span style="color:yellow">机制4：切屏 </span> 一种引导类型的切屏，发动后需要快速找到带绿圈的正方体固定切屏位置；<img src="./img/娅内默2阶段/切屏1.png" alt="切屏1">可以根据角色身上的锁链来进行定位，其他的就是常规切屏了；款式很多，可以参考拉尔戈的切屏机制，在恰当时候进入安全区；<img src="./img/娅内默2阶段/切屏2.png" alt="切屏2">

## <span style="color:red">海伊徳【2阶段】</span>

  - ### <span style="color:yellow">机制1：搭建安保系统 </span> 类似巴卡尔拔剑，奶站中间的篮圈，保证自己不死就行，死了的话机制直接炸；<img src="./img/海伊徳2阶段/搭建安保系统1.png" alt="搭建安保系统1">其他人会被传送到周围的小房间，传到周围小房间的需要破坏所有房间的炮塔；<img src="./img/海伊徳2阶段/搭建安保系统2.png" alt="搭建安保系统2"> 普通模式有4个，分布在角落4个房间，困难8个，全图破坏之后相邻的两个房间会发生秒杀爆炸；推荐所有人集中在一起后，开始一个个破坏；<img src="./img/海伊徳2阶段/搭建安保系统3.png" alt="搭建安保系统3">

  - ### <span style="color:yellow">机制2：噩梦具象化 </span> 开始机制后，玩家被传送到小黑屋内；<img src="./img/海伊徳2阶段/噩梦具象化1.png">正面朝向区域可以打灯，BOSS会在黑暗中（灰雾的位置）逐步接近；被灯光找到，BOSS就会变木头人；<img src="./img/海伊徳2阶段/噩梦具象化2.png" alt="噩梦具象化2"> 随后角色身上出现篮圈，需要引导篮圈攻击房间的科学家；<img src="./img/海伊徳2阶段/噩梦具象化3.png" alt="噩梦具象化3"> 数字清零，则机制成功。

  - ### <span style="color:red">机制3：目标指定【困难独有】 </span> 机制开始后，点名一个队员连续攻击；被点名的需要成功格挡紫色攻击（带图标的）3次；其他人把激光引开，不要干扰到被点名的队员；<img src="./img/海伊徳2阶段/目标指定1.png" alt="目标指定1"> 成功格挡完3次，则机制成功。

## <span style="color:red">人造神纳波尔【2阶段】</span>
 
  - ### <span style="color:yellow">机制1：内心世界 </span> 如果BOSS的白色圈攻击没有格挡到，那么就会进入这个机制；到了内心世界后，每次格挡成功会减少头顶的读条时间；<img src="./img/人造神2阶段/内心世界.png" alt="内心世界">清空之后就可以脱离内心世界。

  - ### <span style="color:yellow">机制2：内心的绝望 </span> BOSS的承伤强制变为0%，进入1分30秒的绝望时间；<img src="./img/人造神2阶段/内心的绝望1.png" alt="内心的绝望1">每格挡成功1次，BOSS的承伤+2%；绝望时间结束后，根据之前格挡成功次数计算当前的承伤率。

  - ### <span style="color:yellow">机制3：注定的毁灭 </span> 地图中生成白色小球，需要站到紫圈内进行格挡，没被挡道的球会被BOSS吸收转化为护盾；<img src="./img/人造神2阶段/注定的毁灭1.png" alt="注定的毁灭1"> 随后地图中间生成紫圈，在紫圈快到最大时格挡；<img src="./img/人造神2阶段/注定的毁灭2.png" alt="注定的毁灭2">随后进入QTE环节，这个会玩的上去玩，不会玩的就别进紫圈挡了，直接在红圈挡；<img src="./img/人造神2阶段/注定的毁灭3.png" alt="注定的毁灭3">QTE的玩家格挡成功后，在绿色区域按下 x，然后疯狂按空格【个人体验：手速慢的话还得两个手去一起按】；<img src="./img/人造神2阶段/注定的毁灭4.png" alt="注定的毁灭4">这个时候，雾神出来帮你，就表示机制成功了，成功后该玩家会获得20%的增伤，持续30秒；<img src="./img/人造神2阶段/注定的毁灭5.png" alt="注定的毁灭5">这之后会读毁灭条，毁灭条满了如果护盾还没有打掉就会全灭。<img src="./img/人造神2阶段/注定的毁灭6.png" alt="注定的毁灭6">

  - ### <span style="color:yellow">冰：天气机制 </span>
    
    - #### <span style="color:yellow">冰机制1：冰冷的安息 </span> 地图中出现3个篮球，1个红球；需要玩家拾取红球后，再沿路捡完篮球；<img src="./img/人造神2阶段/冰/冰冷的安息1.png" alt="冰冷的安息1">但是拾取红球后，移速会变得很低，还伴随大量攻击；<img src="./img/人造神2阶段/冰/冰冷的安息2.png" alt="冰冷的安息2"> 且头顶还有一个倒计时读条，读条结束就会被冻住，球会掉下来；<img src="./img/人造神2阶段/冰/冰冷的安息3.png" alt="冰冷的安息3">其他人需要帮助捡球的人进行格挡；球掉落了就需要其他人来接力捡球了，非常考验配合；

    - #### <span style="color:yellow">冰机制2：冰冷的终结 </span> 地图中出现大篮圈和篮球，先躲篮圈，不然吃到就变僵尸；<img src="./img/人造神2阶段/冰/冰冷的终结1.png" alt="冰冷的终结1">躲完就把4个篮球 推到冰刺上消除掉；小篮球2个人各捡一个，捡完贴贴就可以消除掉；<img src="./img/人造神2阶段/冰/冰冷的终结2.png" alt="冰冷的终结2">在地面出现紫圈的时候，及时格挡，如果没挡道，BOSS迟到生成的紫球会发动全屏攻击；<img src="./img/人造神2阶段/冰/冰冷的终结3.png" alt="冰冷的终结3">上述机制完成越多，BOSS承伤越高。

  - ### <span style="color:yellow">火：天气机制 </span>
  
    - #### <span style="color:yellow">火机制1：灾害 </span> 地图中出现带血条的红球，同时点名玩家发动红圈攻击；<img src="./img/人造神2阶段/火/灾害1.png" alt="灾害1"> 被点名的引红圈攻击红球，直到血条清空为止；其他人减小篮球，撞两边的火墙，扩大安全区；<img src="./img/人造神2阶段/火/灾害2.png" alt="灾害2">机制一共持续三轮，读条清空后结束。
    
    - #### <span style="color:yellow">火机制2：寂静的安息 </span> 地图中出现巨型石柱，石柱有血条；随后会点名2个人发动火焰攻击（身上会有标记）；<img src="./img/人造神2阶段/火/寂静的安息1.png" alt="寂静的安息1">被点名的玩家各找1根柱子钻进去，躲避接下来的火焰攻击；<img src="./img/人造神2阶段/火/寂静的安息2.png" alt="寂静的安息2"> 攻击结束后会出现石块，捡起石块撞石柱，会给石柱回复血量；<img src="./img/人造神2阶段/火/寂静的安息3.png" alt="寂静的安息3"> 如果被点名的人是大红圈，那么就把红圈往下引，远离石柱；<img src="./img/人造神2阶段/火/寂静的安息4.png" alt="寂静的安息4"> 随后会伴随陨石攻击，被陨石点到的找根血量最少的柱子往里钻；<img src="./img/人造神2阶段/火/寂静的安息5.png" alt="寂静的安息5">在寂静的安息读条结束后存活，则机制成功。
    
    - #### <span style="color:yellow">火机制3：锁定火墙 </span> 机制开始会锁定一人，在他的左右两侧生成火墙；<img src="./img/人造神2阶段/火/锁定火墙1.png" alt="锁定火墙1">其他人跟着他一起躲切线就好。
    
  - ### <span style="color:red">雷：天气机制【困难独有】 </span>

    - #### <span style="color:red">雷机制1：无可避免的归宿 </span> 地面会变成湖面，能看到角色的倒影，需要找到看不到倒影的位置；<img src="./img/人造神2阶段/雷/无可避免的归宿1.png" alt="无可避免的归宿1"> 迟到黄球后，引导红圈攻击该位置；成功后，BOSS出现，并赋予玩家黄圈；<img src="./img/人造神2阶段/雷/无可避免的归宿2.png" alt="无可避免的归宿2">其中两个人的全会有黄色射线；<img src="./img/人造神2阶段/雷/无可避免的归宿3.png" alt="无可避免的归宿3">引导射线攻击BOSS，成功4次机制成功；但需要注意，带射线的圈不能和其他圈重叠，否则会直接失效；<img src="./img/人造神2阶段/雷/无可避免的归宿4.png" alt="无可避免的归宿4">通用解法：没线战两边，有线站中间，射线打BOSS，同时注意：中间的两个人不要将圈重叠。<img src="./img/人造神2阶段/雷/无法避免的归宿5.png" alt="无法避免的归宿5">
